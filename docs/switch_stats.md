# Switch Stats (Metrics)

## Generic Fields

  * **rb**: RX Bytes
  * **tb**: TX Bytes
  * **rtp**: RX Total Packets
  * **ttp**: TX Total Packets
  * **rbp**: RX Broadcasts
  * **tbp**: TX Broadcasts
  * **rmp**: RX Multicasts
  * **tmp**: TX Multicasts
  * **rup**: RX Unicasts
  * **tup**: TX Unicasts

## Mapping for CSS610

  * **i01**: rb
  * **i0f**: tb
  * **i23**: rtp
  * **i24**: ttp
  * **i07**: rbp
  * **i14**: tbp
  * **i08**: rmp
  * **i13**: tmp
  * **i05**: rup
  * **i11**: tup
